# Réviseur TSSR — RNCP 37682

Site de révision interactif pour le titre professionnel **Technicien Supérieur Systèmes & Réseaux** (RNCP 37682).

Un seul fichier autonome (`index.html`), sans dépendance externe : fiches expliquées « en clair » avec analogies, schémas, page mémo, antisèche imprimable, plan de révision, et **132 QCM** à correction détaillée.

## Contenu
- **2 blocs / 9 compétences** officielles + une fiche transverse Microsoft 365
- **41 encadrés « 💡 En clair »** (explications simples avec analogies)
- **31 schémas** intégrés (SVG, s'adaptent au thème clair/sombre)
- **🧠 Mémo visuel** · **🎯 Antisèche imprimable** · **📅 Plan 3 jours**
- **132 QCM** avec correction (pourquoi c'est juste, pourquoi le piège est faux)

## Déploiement sur Vercel
1. Sur [vercel.com](https://vercel.com) → **Add New… → Project**
2. Importer ce dépôt GitHub
3. Framework Preset : **Other** — rien à configurer (racine, pas de build)
4. **Deploy** → l'URL sert directement `index.html`

## Local
Ouvre simplement `index.html` dans un navigateur (fonctionne hors-ligne).
